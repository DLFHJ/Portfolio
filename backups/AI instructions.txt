Allowed layout vocabulary (only These should be valid)

Section-level layouts
full-bleed        → spans entire viewport width
editorial-split  → text + image pulled to opposite grid edges
centered         → content centered within grid
offset           → asymmetric placement for visual tension
grid             → repetitive content (galleries only)

Item roles
text
image
video
embed

Width tokens (semantic, not numeric)
small
medium
large
full

Alignment tokens
left
center
Right


The app defines a single global grid (12 columns) for the MODALS (Project details).

Section layout decides how items map to the grid.

Item width maps to predefined column spans.

Item align maps to left/center/right grid alignment.

JSON must never contain raw CSS values or grid column numbers.

Galleries are the only sections allowed to define columns.
