About

Foto © Maximilian Lottmann

I specialize in Interaction Design and bridging the gap between digital and physical spaces. My work often explores Generative Art, focusing on creating immersive environments that react to human presence.

With a background in User Research, I ensure that every visual spectacle is grounded in solid usability principles. Whether it's a large-scale Installation or a mobile app, the goal is always to reduce cognitive load and spark delight.

Skills

A mix of creative coding, classic design tools, and hardware prototyping allows me to build End-to-End prototypes.

Design & Research: UX Design + Interaction Design + User Research + Usability Testing + A/B Testing + User Flows + Wireframing + Konzeption + Design Thinking
Software & Tools: Figma + Sketch + Marvel App + Adobe Creative Suite (Photoshop, Illustrator) + Maya + Blender
Technische Fähigkeiten: HTML + CSS + JavaScript + Java + C# + C++ + Game Engines (z.B. Unity) + Hardware-Prototyping (z.B. Arduino)
